# Discord Bot

A Python Discord bot thta handles server commands and events.

## Features
- Command system for users
- Event handling for server activities
- Modular and easy to expand

## Setup
1. Clone the repository: https://github.com/adelreal707/Custom-Python-Discord-Bot.git
2. Install dependencies: pip install -r requirements.txt
3. Create a '.env' file in the project root with yoru token: DISCORD_TOKEN=YOUR_BOT_TOKEN_HERE
4. Run the bot: pyhton bot.py

## Notes
- Make sure '.env' is **not committed** to GitHub to keep your token secure.
